# Mail Pilot Documentation Site

Technical architecture documentation for Mail Pilot — an AI-native email client built with Next.js, CopilotKit, and the Gmail API.

## Development

```bash
npm install
npm run start
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

The documentation site is built with [Docusaurus](https://docusaurus.io/).
